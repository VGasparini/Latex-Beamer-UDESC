\section{Introdução}
\begin{frame}
    \frametitle{O que é \LaTeX ?}
    \LaTeX é uma linguagem para formatação de textos.
    Geralmente utilizada na elaboração de textos/artigos científicos, porém
    também pode ser utilizada para elaboração de qualquer tipo de material
    tipográfico, como livros, posters, apresentações...
\end{frame}

\begin{frame}
    \frametitle{Uma MUITO breve história}
    \begin{enumerate}
    \item Donald Knuth em 1982 (nosso bom velhinho) criou a linguagem tipográfica \TeX.
    \item Posteriormente Leslei Lamport criou o \LaTeX como conjunto padrão de macros para o \TeX.
    \end{enumerate}
    \LaTeX $\:$ é uma interface fácil e padrão para o \TeX.
\end{frame}

\begin{frame}
    \frametitle{Motivações e vantagens}
    \begin{itemize}
    \item Permiti que você somente se preocupe com o conteúdo do seu texto: Separação do conteúdo e do estilo.
    \item Requer pouco recurso do sistema: você só precisa rodar um editor de texto e eventualmente
      compilar os arquivos.
    \item Escrever fórmulas matemáticas é bastante prático.
    \item Um arquivo \LaTeX é arquivo de texto. Nada é portável que texto puro e pode utilizar controle de versão.
    \item Gerenciamento bibliográfico é fácil e formatação das referências bibliográficas é automático.
    \item Referenciar partes do texto é prático.
    \item Existem ferramentas como TikZ para produzir ilustrações de alta qualidade.
    \item \LaTeX é Software Livre.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Desvantagens}
    \begin{itemize}
    \item Requer aprender uma linguagem e não é só clicar em botões.
    \item Se a formatação do documento não é algo relevante, então pode ser mais prático utilizar um Word da vida.
    \item Algumas vezes é bem difícil encontrar o erro sintático, pois as mensagens de erros são grandes e complexas.
    \end{itemize}
\end{frame}

\section{Primeiros passos}
\begin{frame}
    \frametitle{Como utilizar \LaTeX?}
    \begin{enumerate}
    \item Editor de texto (vim, nano, gedit, notepad...) $+$ compilador \LaTeX (pdflatex, lualatex, xelatex...).
    \item Editores específicos para \LaTeX: Kile, Latexila, Texworks...
    \item Editor online: Overleaf e Sharelatex.
    \end{enumerate}
\end{frame}

\begin{frame}[fragile]
    \frametitle{O que é cada arquivo?}
    \begin{itemize}
    \item \textbf{.tex}: Principal arquivo com todo o conteúdo do texto.
    \item \textbf{.sty}: Arquivo de pacote. Incluído no texto com \lstinline{\usepackage}.
    \item \textbf{.cls}: Arquivo de estilo. Incluído no texto com \lstinline{\documentclass}.
    \item \textbf{.bib}: Arquivo com as informações para as referências bibliográficas.
    \end{itemize}

    \begin{itemize}
    \item \textbf{.log}: Informações detalhes do que aconteceu durante a ultima compilação.
    \item \textbf{.toc}: Arquivo de cabeçalhos, lido durante a próxima compilação.
    \item \textbf{.aux} e \textbf{.out}: Armazenas algumas informações que podem ser utilizadas nas próximas compilações.
    \end{itemize}
\end{frame}

\section{Estrutura de um arquivo \textbf{.tex}}
\begin{frame}[fragile]
    \frametitle{Preâmbulo}
    Tudo entre \lstinline|\documentclass| e \lstinline|\begin{document}|.
    Definição de pacotes e classes a serem utilizados, e outras informações. Isso é particular de cada template.
    \begin{itemize}
    \item \lstinline|\documentclass[opções]{estilo}|.
        \begin{itemize}
          \item opções: fonte, tamanho da fonte, tipo de papel... 
          \item estilo: article, report, book, beamer...
        \end{itemize}
    \item \lstinline|\usepackage[opções]{nome do pacote}|.
        \begin{itemize}
          \item opções: depende das opções do pacote.
        \end{itemize}
    \item Dados como autor, e-mails, endereços...
    \end{itemize}
\end{frame}
