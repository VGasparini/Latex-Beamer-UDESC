\section{Introdução}
\begin{frame}
\frametitle{Computador Clássico}
\begin{itemize}
    \item A computação convencional utiliza dados armazenados em bits: 0 ou 1 (exclusivamente).
    \item Um dado de 3 bits representa ao mesmo tempo apenas um dos seus 8 possíveis estados.
    \item Transistores (interruptores) são combinados para formar portas lógicas, que
          são combinadas para formar circuitos digitas (unidades de processamento: aritmética, lógica etc).
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Conceitos Básicos de Quântica}
\begin{itemize}
  \item A \textbf{Superposição quântica}: uma partícula pode assumir simultaneamente proporções entre dois estados.\\
  Ex: A polarização de um fóton pode ser qualquer proporção entre vertical ou horizontal. Mas uma vez que 
  observado, o seu estado colapsa (com base na probabilidade das proporções) para um dos extremos.
  \item A superposição quântica pode ser utilizada para representar informação binária: \textbf{qubit}.
  \item A computação quântica utiliza dados armazenados em qubits: 0 - 1 (qualquer proporção de ambos estados).
  \item Um dado de 3 qubits representar ao mesmo tempo todos os 8 possíveis estados.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Representação dos Qubits}
\begin{itemize}
  \item Os dois estados base de um qubit são representados pelos vetores
    \begin{align}
      |0 \: = \begin{bmatrix} 
              1 \\ 
              0 \\ 
            \end{bmatrix}
      |1 \: = \begin{bmatrix} 
              0 \\ 
              1 \\ 
            \end{bmatrix}
    \end{align}
  \item Para completamente representar o estado de um qubit são necessários dois números para descrever as duas proporções das duas bases:
    \begin{align}
      \Psi = a|0 + b|1.
    \end{align}
  \item Para representar o estado de dois qubits são necessários quatro números:
    \begin{align}
      \Psi = a|00 + b |01.
    \end{align}
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Portas Quânticas}
\begin{itemize}
  \item Portas quânticas são análogas as portas lógicas binárias.
\end{itemize}
\end{frame}
