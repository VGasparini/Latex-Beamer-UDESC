\section{Introdução}
\begin{frame}
    \frametitle{O que é \LaTeX ?}
    \LaTeX é uma linguagem para formatação de textos.
    Geralmente utilizada na elaboração de textos/artigos científicos, porém
    também pode ser utilizada para elaboração de qualquer tipo de material
    tipográfico, como livros, posters, apresentações...
\end{frame}

\begin{frame}
    \frametitle{Uma MUITO breve história}
    \begin{enumerate}
    \item Donald Knuth em 1982 (nosso bom velhinho) criou a linguagem tipográfica \TeX.
    \item Posteriormente Leslei Lamport criou o \LaTeX como conjunto padrão de macros para o \TeX.
    \end{enumerate}
    \LaTeX $\:$ é uma interface fácil e padrão para o \TeX.
\end{frame}

\begin{frame}
    \frametitle{Motivações e vantagens}
    \begin{itemize}
    \item Permiti que você somente se preocupe com o conteúdo do seu texto: Separação do conteúdo e do estilo.
    \item Requer pouco recurso do sistema: você só precisa rodar um editor de texto e eventualmente
      compilar os arquivos.
    \item Escrever fórmulas matemáticas é bastante prático.
    \item Um arquivo \LaTeX é arquivo de texto. Nada é portável que texto puro e pode utilizar controle de versão.
    \item Gerenciamento bibliográfico é fácil e formatação das referências bibliográficas é automático.
    \item Referenciar partes do texto é prático.
    \item Existem ferramentas como TikZ para produzir ilustrações de alta qualidade.
    \item \LaTeX é Software Livre.
    \end{itemize}
\end{frame}

\begin{frame}
    \frametitle{Desvantagens}
    \begin{itemize}
    \item Requer aprender uma linguagem e não é só clicar em botões.
    \item Se a formatação do documento não é algo relevante, então pode ser mais prático utilizar um Word da vida.
    \item Algumas vezes é bem difícil encontrar o erro sintático, pois as mensagens de erros são grandes e complexas.
    \end{itemize}
\end{frame}

\section{Estrutura e comandos básicos}
\begin{frame}
    \frametitle{Como utilizar \LaTeX?}
    \begin{enumerate}
    \item Editor de texto (vim, nano, gedit, notepad...) $+$ compilador \LaTeX (pdflatex, lualatex, xelatex...).
    \item Editores específicos para \LaTeX: Kile, Latexila, Texworks...
    \item Editor online: Overleaf e Sharelatex.
    \end{enumerate}
\end{frame}

\begin{frame}[fragile]
    \frametitle{O que é cada arquivo?}
    \begin{itemize}
    \item \textbf{.tex}: Principal arquivo com todo o conteúdo do texto.
    \item \textbf{.sty}: Arquivo de pacote. Incluído no texto com \lstinline{\usepackage}.
    \item \textbf{.cls}: Arquivo de estilo. Incluído no texto com \lstinline{\documentclass}.
    \item \textbf{.bib}: Arquivo com as informações para as referências bibliográficas.
    \end{itemize}

    \begin{itemize}
    \item \textbf{.log}: Informações detalhes do que aconteceu durante a ultima compilação.
    \item \textbf{.toc}: Arquivo de cabeçalhos, lido durante a próxima compilação.
    \item \textbf{.aux} e \textbf{.out}: Armazenas algumas informações que podem ser utilizadas nas próximas compilações.
    \end{itemize}
\end{frame}

\begin{frame}[fragile]
    \frametitle{Preâmbulo}
    Tudo entre \lstinline|\documentclass| e \lstinline|\begin{document}|.
    Definição de pacotes e classes a serem utilizados, e outras informações. Isso é particular de cada template.
    \begin{itemize}
    \item \lstinline|\documentclass[opções]{estilo}|.
        \begin{itemize}
          \item opções: fonte, tamanho da fonte, tipo de papel... 
          \item estilo: article, report, book, beamer...
        \end{itemize}
    \item \lstinline|\usepackage[opções]{nome do pacote}|.
        \begin{itemize}
          \item opções: depende das opções do pacote.
        \end{itemize}
    \item Dados como autor, e-mails, endereços...
    \end{itemize}
\end{frame}

\begin{frame}[fragile]
    \frametitle{Comandos Principais:}
  \begin{itemize}
    \item \lstinline|\section|, \lstinline|\subsection|, \lstinline|\subsubsection|.
    \item Quebrar em parágrafos: Linha em branco ou utilizar \lstinline| \\ |. 
    \item Estilo de fonte: \lstinline|\textbf{texto em negrito}| e \lstinline|\textbf{texto em itálico}|.
    \item Tamanho da fonte: \lstinline|\tiny{texto}|, \lstinline|\small{texto}|, \lstinline|\large{texto}|...
    \item Nota de rodapé: \lstinline|\footnote{Uma nota de rodapé}|.
    \item Referenciar o partes do texto: \lstinline|{\label{chave}| e \lstinline|\ref{chave}|.
    \item Aspas duplas: \lstinline|``texto''|.
  \end{itemize}
\end{frame}

\begin{frame}[fragile]
    \frametitle{Modo matemática}
  \begin{itemize}
    \item O modo matemática \textit{inline} é um escopo especial entre \lstinline|$ $| para fórmulas matemáticas.
    \item \lstinline|$ 3 + \pi \cdot \sum_{i=0}^{10} i^2 $|
    \item $ 3 + \pi \cdot \sum_{i=1}^{10} i^2 $
    \item Para fazer uma fórmula/equação em próprio espaço utilize o ambiente align. Para não dar um número
      para cada equação utilize align*.
  \end{itemize}

\begin{columns}
    \begin{column}{0.48\textwidth}
\begin{lstlisting}
  \begin{align*}
    fat \: 3 & = fat 2 * fat 1 \\
    fat \: 2 & = fat 1 * fat 0 \\
    fat \: 1 & = 1 \\
    fat \: 0 & = 1 
  \end{align*}
\end{lstlisting}
    \end{column}

\section{Objetos flutuantes}

\begin{frame}[fragile]
    \frametitle{Imagens}
  \begin{itemize}
    \item Objetos flutuantes são imagens, tabelas, algoritmos...
    \item \LaTeX tenta evitar que objetos flutuantes causem páginas que partes em branco, assim
      onde está o código referente ao objeto flutuante não será necessariamente o local após a compilação.
    \item Parâmetros para \textit{permissão} de posicionamento:
    \begin{itemize}
      \item h = here , t = top , b = bottom , ! = sugestão para boa posição.
    \end{itemize}
  \end{itemize}
\end{frame}

\begin{frame}[fragile]
  Um bom exemplo de como inserir uma figura:
  \begin{lstlisting}
     \begin{figure}[th!]
       \centering
       \includegraphics[width=.3\textwidth]{fig1.jpg}
       \caption{A typical figure}
       \label{fig:exampleFig1}
     \end{figure}
  \end{lstlisting}

       \label{fig:exampleFig1}
     \end{figure}
\end{frame}

\begin{frame}[fragile]
    \frametitle{Tabelas}
  \begin{itemize}
    \item O ambiente table serve para criar tabelas. 
    \item Utiliza os mesmos parâmetros para posicionamento que as figuras, também é possível adicionar caption e label.
    \item Para definir de fato uma tabela utiliza-se ambiente tabular.
    \item O alinhamento do texto da coluna é definido pelos paramentos: l para esquerda; c para centralizado e r para direita. 
      \lstinline|\begin{tabular}{lllll}| são quatro colunas justificadas para a esquerda.
    \item \lstinline|&| separa as colunas e  \lstinline| \\ | as linhas.
    \item Para tornar visível a linha que separa as colunas adicione $|$ (pipes) entre os parâmetros que justificam as colunas. 
    \item Para tornar visível a linha que separa as linhas utilize \lstinline|\hline| após o \lstinline| \\ | de pular linha.
  \end{itemize}
\end{frame}

\begin{frame}[fragile]
  Um bom exemplo de como inserir uma tabela:

\section{Referências bibliográficas}

\begin{frame}[fragile]
    \frametitle{Referências bibliográficas}
  \begin{itemize}
    \item As referências são armazenadas em arquivos \textbf{.bib} e devem ser compiladas com o bibtex. 
    \item Geralmente uma compilação completa é:  pdflatex article.tex ; bibtex referecias.bib ; pdflatex article.tex .
    \item Uma referência bibliográfica no arquivo .bib é uma estrutura com várias informações: Começa com um @ e o tipo de documento (Book, InCollection, Article..).
    \item A primeira expressão da referência é a chave.
    \item Para citar uma referência use \lstinline|\cite{knuth:84}|.
  \end{itemize}
  \begin{lstlisting}
@Book{knuth:84,
  author = 	 {Donald E. Knuth},
  title = 	 {The {\TeX} Book},
  publisher = 	 {Addison-Wesley},
  year = 	 {1984},
  edition = 	 {15th}
}
  \end{lstlisting}
\end{frame}
