\section{A Universidade}
\begin{frame}
\frametitle{O que é a universidade?}
    \begin{itemize}
        \item Qual o papel primordial das universidades?
        \item O artigo 207 da constituição garante o chamado tripé da universidade, que
            é a indissociabilidade entre ensino, pesquisa e extensão.
    \begin {itemize}
        \item O ensino acontece por meio das aulas, exercício dos docentes.
        \item A pesquisa é aplicação dos conhecimentos básicos, passado pelos docentes, com a finalidade principal de produzir algo novo.
        \item A extensão busca articular com a sociedade os conhecimentos desenvolvidos por meio da pesquisa e do ensino.
    \end{itemize}
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Para que entrar numa universidade?}
    \begin{itemize}
        \item O mito:a universidade serve exclusivamente para  alimentar o mercado de trabalho.
        \item Consequências:
                \begin{itemize}
                   \item Criação de mais universidades particulares.
                   \item Baixo desenvolvimento de pesquisas.
                   \item ''Venda´´ de diplomas.
                   \item Fim da meritocracia.
                \end{itemize}
    \end{itemize}
\end{itemize}
\end{frame}
